### Option 1: Vercel CLI (Recommended)

```bash

# 2. Navigate to project folder

# 3. Deploy
```

---

### Option 2: Vercel Website (No CLI needed)

1. **Go to**: https://vercel.com/new
2. **Drag & drop** the entire `f1-project-website` folder
3. **Click Deploy**

Done! Your site is live.

---

### Option 3: GitHub + Vercel (Best for ongoing updates)

1. **Create GitHub repo** and upload these files
2. **Go to**: https://vercel.com/new
3. **Import** your GitHub repository
4. **Click Deploy**

Now every push to GitHub auto-deploys!
